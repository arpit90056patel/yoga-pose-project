# 🧘‍♂️ Yoga Pose Detection System using Deep Learning

## 📌 Overview

The Yoga Pose Detection System is a deep learning-based web application designed to classify yoga poses from images and videos. The system helps users identify whether a yoga posture is performed correctly without requiring a physical trainer.

This project demonstrates the practical implementation of **Computer Vision** and **Deep Learning (CNN)** in the fitness domain.

---

## 🎯 Objectives

* To build an intelligent system for yoga pose classification
* To support both image and video input
* To provide a simple and user-friendly web interface
* To apply deep learning techniques for real-world problem solving

---

## 🚀 Features

* 📷 Image-based yoga pose detection
* 🎥 Video-based pose detection
* ⚡ Fast and accurate predictions
* 💻 Clean and responsive web interface
* 🧠 Deep learning model integration

---

## 🏗️ System Architecture

User Input (Image/Video) → Flask Backend → Deep Learning Model → Prediction Output → Result Display

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/arpit90056patel/yoga-pose-project.git
cd yoga-pose-project
```

### 2. Create virtual environment

```
python -m venv .venv
```

### 3. Activate environment

```
.venv\Scripts\activate
```

### 4. Install dependencies

```
pip install -r requirements.txt
```

### 5. Run the application

```
python app.py
```
